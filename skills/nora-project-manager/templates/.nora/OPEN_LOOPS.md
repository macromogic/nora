# Open Loops

## Technical

- What is the current core technical claim?
- Which implementation pieces are complete?
- Which assumptions need to be re-validated?

## Experiments

- Which results are already available?
- Which results are reproducible?
- Which figures/tables have unclear provenance?

## Writing

- What is the current target venue?
- What is the current story?
- Which sections are stale?

## Project management

- Who is involved?
- What is the next realistic milestone?
- What caused the project to stall?
