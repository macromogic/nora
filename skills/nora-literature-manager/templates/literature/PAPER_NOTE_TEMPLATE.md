# Paper Note Template

Copy this structure to `.nora/literature/PAPER_NOTES/<id>.md`, where `<id>` is the paper's `papers.yaml` id (see `workflows/paper-note.md`). Status and roles live in `papers.yaml` (change them via `nora literature mark`), so the note only records identity plus what structured state can't hold. Do not invent values for fields you don't have information for — write `unknown` or `not yet read`.
